Zero-Day Exploit

Um Zero-Day Exploit é uma vulnerabilidade de segurança desconhecida para o desenvolvedor do software ou sistema, e que, portanto, ainda não tem correção disponível. O termo "zero-day" se refere ao fato de que os responsáveis pela segurança tiveram zero dias para resolver o problema antes que ele fosse explorado. Hackers e cibercriminosos podem usar essas falhas para invadir sistemas, roubar dados ou causar danos antes que uma correção seja lançada.
